中國清華紫光集團旗下的紫光國芯半導體（UniIC）於 3 日在晶圓代工廠格羅方德（Globalfoundries）舉行的「全球技術論壇」（GTC）當中宣布，發表新產品「+ PHY IP」。 該產品是一款以格羅方德 12LP 平台為基礎的第 6 代繪圖用雙倍數據傳輸率（GDDR6）記憶體控制器（MC）。與市面現有產品相比，新一代解決方案在功率、面積和成本上均有顯著改善，足以因應人工智慧（AI）和運算應用不斷增長的需求。

紫光國芯表示，旗下的 GDDR6 IP 所包含的記憶體控制器，不但可供配置，亦符合 DFI3.1 和 AMBA AXI4.0 等規範，使設計人員得以創造出具有最佳化延遲和頻寬的 GDDR6 控制器，進而滿足顯示卡、遊戲主機及 AI 運算等高效能應用的需求。
另外，紫光國芯的 PHY 是專為提高功率和效能而設計，可提供高達 12Gbps/13Gbps/14Gbps/16Gbps 的資料傳輸率，並與符合 JECEC 標準的 JEDEC250 和 DFI3.1 相容。PHY 還提供嵌入式高效能 PLL，以符合時序相關規格。通過矽晶認證的 IP 在與主流 GDDR6 裝置整合後，可於 12Gbps/14Gbps/16Gbps 的速度下運行。在 16Gbps 的運 行速度下，每 DQ 的最大功耗小於 4mW/Gbps。
紫光國芯總經理任奇偉指出，人工智慧和網路市場正在迅速成長，連帶刺激了外界對記憶體頻寬的需求。在藉助格羅方德 12LP 製程的情況下，紫光國芯的 GDDR6 IP 效能將顯著提升，並大大推動 GDDR6 應用的發展。而格羅方德生態系統合作夥伴關係副總裁 Mark Ireland 也表示，紫光國芯的 GDDR6 IP 使格羅方德 12LP 製程功能更加完善，為應對高頻寬記憶體的爆炸性增長做足了充分準備。透過這次合作，拓展了 IP 和服務範圍，進而讓共同顧客有辦法提供功能更強大且具成本效益的解決方案。目前，紫光國芯的 GDDR6 MC + PHY IP 現已上市。
